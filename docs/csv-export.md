# CSV出力機能

採点データをCSVファイルに出力できます。Excelで開くことができます。

## 使い方

```bash
# 精密採点Ai
npm run export:csv:ai

# 精密採点Ai Hearts
npm run export:csv:hearts

# 精密採点DX-G
npm run export:csv:dxg
```

## 出力内容

CSVファイルには以下の32項目が含まれます：

### 基本情報
- 採点ID
- 曲名
- アーティスト
- 演奏日時
- 総合点

### レーダーチャート
- 音程
- 安定性
- 表現力
- リズム
- ビブラート/ロングトーン

### ビブラート
- ビブラート回数
- ビブラート上手さ
- ビブラート秒数
- ビブラートタイプ

### 表現テクニック
- こぶし
- しゃくり
- フォール
- 抑揚
- ハンマリング
- アクセント
- エッジボイス
- ヒーカップ

### DX-G裏加点
- ボーナスタイプ（音程/ビブラート/表現力）
- ボーナスポイント

### その他
- ロングトーンスキル
- リズムタイミング
- キー設定

### 全国平均
- 全国平均総合点
- 全国平均音程
- 全国平均安定性
- 全国平均表現力
- 全国平均リズム

## 出力例

```csv
採点ID,曲名,アーティスト,演奏日時,総合点,音程,安定性,...
2522015,地球最後の告白を,kemu feat.GUMI,2025-11-21 19:54:11,89.229,83,79,...
```

## ファイル形式

- **エンコーディング**: BOM付きUTF-8（Excelで正しく開ける）
- **区切り文字**: カンマ（,）
- **ファイル名**: `dam_{タイプ}_{タイムスタンプ}.csv`
  - 例: `dam_ai_2026-02-04T03-50-09.csv`
- **保存先**: `output/` フォルダ

## Excelで開く

1. CSV出力コマンドを実行
2. `output/` フォルダ内のCSVファイルをダブルクリック
3. Excelで自動的に開きます

**注意**: BOM付きUTF-8形式なので、日本語が正しく表示されます。
